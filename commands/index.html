SERVER COMMANDS

伺服器常用指令與工具清單（點擊指令區塊可直接複製）

✔ 指令已成功複製到剪貼簿！

🛡️ 1. 方塊安全防盜系統 (Bolt)

這款外掛負責保護世界中的箱子與門，防範偷竊。所有玩家皆可免費使用基礎鎖定。

全玩家
/bolt lock
對著箱子/門按右鍵，直接進行私人鎖定（所有玩家預設可用）。
全玩家
/bolt unlock
對著自己的箱子按右鍵，解除鎖定。
全玩家
/bolt trust <玩家ID>
對著箱子按右鍵，將箱子使用權限共享給朋友。
全玩家
/bolt info
對著任意方塊按右鍵，查詢該方塊是誰鎖定的。
管理員
/bolt admin
管理員專用，開啟最高管理模式（可強制拆除他人鎖定的箱子）。

🎨 2. 身份組與聊天顏色 (LuckPerms)

這款外掛控制玩家名字前方的標籤（Prefix）與名字顏色，用來劃分權限等級。

管理員
/lp editor
產生安全網址，開啟網頁版的視覺化設定面板。
管理員
/lp user <玩家ID> parent set admin
將某人（或自己）提升為管理員組（變更為綠標、黃名）。
管理員
/lp group default meta setprefix "&7[玩家]&f "
一鍵設定新進玩家的格式（灰色 [玩家] 標籤 + 白色名字）。
管理員
/lp group admin meta setprefix "&a[管理員]&e "
一鍵設定管理員的格式（鮮綠 [管理員] 標籤 + 黃色名字）。

💬 3. 基礎指令與私訊系統 (EssentialsX)

這款外掛負責伺服器基礎運作、私密對話、暱稱系統以及後台設定重載。

管理員
/ess reload
修改 config.yml 或 motd.txt 歡迎訊息後，一鍵刷新生效。
全玩家
/msg <玩家ID> <訊息>
發送只有彼此看得到的私密密語（亦可使用快捷指令：/w）。
全玩家
/r <訊息>
快速回覆上一則收到的私訊。
管理員
/nick <玩家ID> <新名字>
幫指定玩家在聊天室換上專屬暱稱（若要清除請輸入 /nick <玩家ID> off）。

⚙️ 4. 麥塊原版內建指令 (Vanilla Gamerule)

控制世界最底層生存規則的通用原版指令，會直接儲存在世界存檔內。

管理員
/gamerule keepInventory true
開啟死亡不掉落（防噴）功能，死後保留所有裝備與經驗值。
找不到任何符合關鍵字的指令！
