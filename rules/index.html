SERVER RULES

Lazy Sheep 伺服器玩家守則與處罰標準

更新時間：2026/07/20 12:00

1. 加入伺服器

在進入伺服器前，必須閱覽全篇。進入遊戲即視同同意本伺服器所有條款。
1-1
進入辦法：
啟動你的 Minecraft Java Edition，點選「多人遊戲」，接著點選「新增伺服器」並在伺服器位置填寫 java.mcsvr.sld.tw，接著點選伺服器兩下。
※ 無論如何，伺服器必須在開啟狀態才可加入（若離線請寄信聯絡管理員）。
1-2
出生點規範：
進入後，你將會生成在一個有山圍繞的村莊，裡面的房子可自行入住。
村莊不可隨意拆除、傷害村民/鐵魔像，違反者進小黑屋！

2. 遊玩與財產安全規範

為了維持友善且長久的生存環境，請共同維護彼此的建築與資產。
2-1
禁止破壞與偷竊 (Griefing & Stealing)：
嚴禁未經同意破壞他人建築、拿取箱子內的任何物資或破壞農作物。個人貴重箱子與門請善用 /bolt lock 指令進行鎖定。
2-2
嚴禁使用外掛與作弊模組 (Cheating / Hacks)：
嚴禁使用任何影響遊戲平衡的外掛程式（例如：X-Ray 透視、KillAura 自動攻擊、Fly 飛行、連點器等）。使用作弊模組者將直接永久封鎖（Permanent Ban）。
2-3
社群互動與聊天規範：
雖然自己人當然要瘋狂罵人，但請保持基本的默契與底線！嚴禁惡意洗屏、騷擾、張貼違法或極端仇恨言論。

3. 伺服器效能與環境維護

由於伺服器為免費主機，請大家維持世界運作順暢。
3-1
限制高頻紅石與卡機裝置：
禁止建造極高頻紅石線路、大型無限重複掉落物裝置或過大規模的動物養殖場。若導致伺服器 TPS 嚴重下降，管理員有權強制拆除。
3-2
保護出生點地形：
請勿在出生點村莊周圍大規模挖坑不填或破壞地形，維護新手村景觀。

4. 處罰機制與申訴

公平執法，維護多數玩家權益。
4-1
處罰層級：
違反規範者視情節輕重處以：口頭警告 ➔ 關入小黑屋 (Jail) ➔ 踢出伺服器 (Kick) ➔ 永久封鎖 (Ban)。
4-2
違規舉報與回報：
若遭遇財產損失或發現違規玩家，請點選首頁「寄信通知管理員」進行檢舉與備份救援。
