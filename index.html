如何開始遊戲？

  • 確認伺服器是否開啟：
    若發現伺服器未連線，請點選上方按鈕聯絡管理員開服。
  • 進入多人遊戲：
    啟動您的 Minecraft Java 版客戶端，點選 「多人遊戲 (Multiplayer)」。
  • 新增伺服器：
    點選 「新增伺服器 (Add Server)」，並在伺服器位址輸入欄貼上複製的網域：
    java.mcsvr.sld.tw
  • 連線冒險：
    點擊 「完成 (Done)」，雙擊伺服器即可直接進入世界！
